Module wayland_backend::sys::client

source ·
Expand description

Client-side implementation of a Wayland protocol backend using libwayland

Entrypoints are:

  • Backend::connect method if you’re creating the Wayland connection
  • Backend::from_foreign_display if you’re interacting with an already existing Wayland connection through FFI.
